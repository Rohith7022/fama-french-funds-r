# Fama-French Funds Analysis
## VCU – Financial Analytics (FIRE 540)

Analysis of investment fund performance through the **Fama-French factor lens** using R and the tidyverse. Evaluates factor exposures, return attribution, and fund performance relative to the FF model benchmarks.

---

## Overview

Applied the Fama-French framework to mutual fund return data to decompose performance into factor contributions (market, size, value) and identify alpha generation relative to systematic risk exposures.

---

## Analysis Includes

- Factor loading estimation for each fund via OLS regression
- Performance attribution: how much of return is explained by FF factors vs. true alpha
- Cross-fund comparison of risk-adjusted performance
- Visualization of factor exposures and return distributions

---

## Tech Stack

`R` `tidyverse` `broom` `ggplot2` `PerformanceAnalytics` `Quarto`
